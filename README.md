# sistema_previsao_academica
Sistema de Previsão de Desempenho Académico. Analisa notas e faltas, gera alertas de risco de reprovação, oferece recomendações personalizadas de estudo e sugere carreiras com base nas aptidões reais. Arquitetura distribuída com microsserviços (FastAPI, SQLite, HTTP) e dashboard Streamlit para monitorar o progresso académico de aprendizagem.
